Eclipse IDE

A feature-rich Java IDE

A full Java IDE

Eclipse IDE is a full integrated, feature rich Java IDE. It is used by professionals all over the software development industry due to its high level of support and excellent community marketplace for plugins. It is also used by many novices thanks to the brilliant quality-of-life features it includes - this is a great starting point for any Java beginner.

Pros/Cons

  • Community supported plugin system

  • Highly customisable

  • Excellent debugging tools

  • High RAM Usage

  • Switching perspectives/workspaces slow

  • Default style somewhat outdated

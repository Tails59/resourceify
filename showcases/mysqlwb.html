MySQL Workbench

Visual database tool for everyone.

The Leading DB-IDE

MySQL Workbench provides facilities for all aspects of database development, creation and management. With built in local server hosting, testing has never been easier.


Pros/Cons

  • 100% Free

  • Visualise your database in diagrams

  • Cross platform

  • No query aborting

  • More complex that what most people need

  • Default style somewhat outdated
